Prototype of automatic content generator using LLMs. The following description is in russian

Тестовый генератор контента:
- сервер запускается через /content_generation_servise/app.py
- клиент - в браузере по адресу http://127.0.0.1:4990/content_generator

Использование:
- Создаем любой набор любых произвольных параметров, задаем значения, отмечаем флагом нужные
и тыкаем в кнопку Generate. контент материализуется в поле ввода снизу
- Все созданные типы параметров, параметры и варианты их значений сохраняются в БД и доступны
для повторного использования
- Результаты генерации тоже сохраняются в базу (вручную. по кнопке Save result). Для одного и того
же сочетания параметров и их значений можно хранить несколько результатов
- Сохраненные результаты по выбранным параметрам можно посмотреть по кнопке Show results.
Обе кнопки - под полем вывода результатов.
- Удалить или отредактировать типы, параметры и варианты значений можно на странице настроек 
(кнопка Settings, левый верхний угол страницы).

Страница настроек:
- Для раскрытия списка дочерних элементов (параметры типа, варианты значений параметра) - кликаем
по полю id нужного элемента.
- для добавления - тыкаем в кнопку Add new element
- для редактирования или удаления - вызываем контекстное меню нужного элемента и выбраем нужный
пункт там

Страница редактирования (добавление и правка):
- вводим нужный текст в соответствующие поля 
 (кроме reference, is_new и полей id - они для справки) и тыкаем Confirm. Или Abort, если передумали
- после Confirm или Abort произойдет автоматический редирект на ту страницу, с которой было вызвано
редактирование

Требования:
- по версии питона и библиотекам всё в requirements.txt. Можно установить из него автоматом через
pip
